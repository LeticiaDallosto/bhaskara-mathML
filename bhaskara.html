Fórmula de Bhaskara utilizando MathML

A fórmula de Bhaskara nada mais é do que uma forma de isolar a incógnita x da equação do 2º grau. Reparem que o ponto de partida da dedução da fórmula é a própria equação do 2º grau, e na sequência, são utilizados artifícios matemáticos a fim de isolar a incógnita x.

Equação quadrática: a x 2 + b x + c = 0 Passo 1: Dividir por a para simplificar a equação x 2 + b a x + c a = 0 Passo 2: Isolar o termo quadrático e o termo linear x 2 + b a x = - c a Passo 3: Completar o quadrado no lado esquerdo da equação x 2 + b a x + ( b 2 a ) 2 = ( b 2 a ) 2 - c a Simplificando o lado esquerdo: x 2 + b a x + b 2 4 a 2 = b 2 - 4 a c 4 a 2 Passo 4: Escrever o lado esquerdo como um quadrado perfeito ( x + b 2 a ) 2 = b 2 - 4 a c 4 a 2 Passo 5: Tomar a raiz quadrada em ambos os lados da equação x + b 2 a = ± b 2 - 4 a c 2 a Passo 6: Isolar x no lado esquerdo da equação x = - b 2 a ± b 2 - 4 a c 2 a Fórmula de Bhaskara completa: x = - b ± b 2 - 4 a c 2 a
